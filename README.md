## The layout

Holding down the left CONTROL key gives you access to the following layer.


         +------+ +------+ +------+ +------+
         | bksp | | del  | | ins  | | pgup |
         |  U   | |  I   | |  O   | |  P   |
         +------+ +------+ +------+ +------+
                   +------+          +------+
                   |  ↑   |          | pgdn |
                   |  K   |          |  ;   |
                   +------+          +------+
    +------+ +------+ +------+ +------+ +------+
    | home | |  ←   | |  ↓   | |  →   | | end  |
    |  N   | |  M   | |  ,   | |  .   | |  /   |
    +------+ +------+ +------+ +------+ +------+
